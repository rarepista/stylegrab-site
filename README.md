# StyleGrab — marketing site

Single-file static site (`index.html`) with a live, working demo of the StyleGrab
inspector (a trimmed port of the real extraction engine). No build step, no deps.

Host free on GitHub Pages: Settings → Pages → Deploy from branch → main / root.
