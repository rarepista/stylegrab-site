What it does

DevTools shows you styles. StyleGrab hands them to you.

Authored CSS, not computed soup

Reads the actual rules the developer wrote — merged by real cascade and specificity — with a rendered-styles fallback when stylesheets are locked.

:hover, @media, @keyframes included

One copy grabs pseudo-classes, pseudo-elements, responsive breakpoints and the animations an element references.

Six output dialects

Clean CSS, Tailwind (utility classes on real HTML, not just a class string), React/JSX, styled-components, Vue, or Svelte — pick per element or per component.

Portable by default

var(--tokens) resolve to real values, inherited typography folds in, and every rule keeps a comment naming which stylesheet it came from.

Pin, edit, measure, screenshot

Pin the panel and edit CSS with live preview on the page. Alt-hover measures pixel distances. Copy any element as a PNG.

WCAG contrast, palette & assets

A contrast-ratio chip on every text element, a full page color palette with usage counts (copy as CSS variables), and an image/SVG collector with real thumbnails.

FAQ

Questions people actually ask.

How is this different from DevTools?

DevTools shows one rule at a time and makes you reassemble the cascade by hand. StyleGrab merges everything that applies to an element — including :hover states, breakpoints and keyframes — into one paste-ready block, then converts it to your framework's dialect.

Is the free tier actually free?

Yes, permanently: hover-inspect, full CSS copy, box-model overlay and the eyedropper never expire and never nag. Pro adds the framework exports, component copy, live editing and extraction tools.

What data do you collect?

None. The extension runs entirely in your browser, makes zero network requests during normal use, and only touches pages where you explicitly invoke it (activeTab permission). Checkout, if you buy, happens through Dodo Payments — a merchant of record that handles cards, UPI, PayPal and taxes on our behalf. We never see your payment details, and there's no account to create on either side.

What happens after I buy?

Dodo Payments emails you a license key automatically. Paste it into StyleGrab's Options page and Pro unlocks instantly — no account, no login. The key re-validates itself every few days in the background; if you're offline, StyleGrab keeps your Pro status rather than locking you out.

Does it work on React / Vue / Tailwind sites?

Yes — it reads the browser's stylesheet model, so the source framework doesn't matter. It also understands modern CSS plumbing: @layer, adopted stylesheets and shadow-root styles.

Copying styles — is that okay?

StyleGrab is a learning and productivity tool, the same way View Source and DevTools are. Use it to study techniques, debug, and move your own work between projects. Respect other people's IP; don't clone whole products.

Is $29 really a one-time payment, or does it turn into a subscription?

One time, full stop. No recurring billing exists in StyleGrab's checkout — there's nothing to cancel later because nothing renews. You get every update to Pro features for as long as you use the extension, for that single payment.

How many browsers can I use my license on?

A license key activates on up to 3 browser profiles at once — enough for a work laptop, a personal machine, and a backup. Remove a key from one in Options to free up a slot for another.

Does it work in Edge or Brave?

Yes — StyleGrab is a standard Manifest V3 extension with no Chrome-only APIs, so it installs and runs identically on any Chromium-based browser (Edge, Brave, Arc, Opera) via the same package.
